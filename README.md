# Teizer Notes — Premium AI Notepad

Персональное приложение для заметок с AI-ассистентом, голосовым вводом и T9.

---

## Запуск (быстрый способ — без сборки .exe)

### Шаг 1 — Установи Node.js
Скачай с https://nodejs.org (LTS версия, 18+)
При установке галочка "Add to PATH" должна быть включена.

### Шаг 2 — Открой папку в терминале
```
Правой кнопкой на папке teizer-notes-app → "Открыть в терминале"
```
Или через PowerShell / CMD:
```
cd C:\путь\до\teizer-notes-app
```

### Шаг 3 — Установи зависимости
```
npm install
```
Подождать 1-2 минуты пока скачается Electron.

### Шаг 4 — Запуск
```
npm start
```

---

## Сборка в .exe (установщик)

```
npm run build
```
Готовый установщик появится в папке `dist/`.

## Возможности

| Функция | Как использовать |
|---------|-----------------|
| **Голосовой ввод** | Кнопка Voice в тулбаре (Ctrl+M) |
| **T9 / автодополнение** | Появляется автоматически при вводе (рус + англ) |
| **AI-ассистент** | Панель справа — нужен API ключ |
| **Категории** | Work / Ideas / Personal / Misc |
| **Теги** | Поле "+ add tag" над редактором |
| **Поиск** | Строка поиска вверху |
| **Автосохранение** | Каждые 600мс, сохраняется локально |

---

## AI-ассистент

1. Нажми "Settings & API Key" внизу сайдбара (Ctrl+,)
2. Вставь API ключ от Anthropic (console.anthropic.com)
3. Нажми Save Settings

Ключ хранится **только локально** на твоём компьютере.

Быстрые команды ассистента:
- **Summarize** — краткое резюме заметки
- **Improve style** — улучшить стиль текста
- **Ideas** — генерация идей по теме заметки
- **To-do list** — создать список задач
- **Fix grammar** — исправить грамматику
- **Translate EN** — перевести на английский

## Горячие клавиши

| Комбинация | Действие |
|-----------|---------|
| `Ctrl+N` | Новая заметка |
| `Ctrl+F` | Поиск |
| `Ctrl+M` | Голосовой ввод вкл/выкл |
| `Ctrl+,` | Настройки |
| `Escape` | Закрыть диалог |

## Где хранятся данные

Windows: `C:\Users\ИМЯ\AppData\Roaming\teizer-notes\teizer_notes.json`

Данные сохраняются автоматически при каждом изменении и при закрытии.
